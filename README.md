# Schlagzeug Verwaltung

Eine WPF-Anwendung zur Verwaltung von Schlagzeugschuelern, Terminen, Zahlungen, Wochennotizen und Drum-Notation.

## Funktionen

- Schuelerverwaltung
- Terminverwaltung
- Zahlungen erfassen, loeschen und zuruecksetzen
- Monatsuebersicht mit Guthaben- und Ausstehend-Berechnung
- Wochennotizen fuer Schueler
- Drum-Notation mit GrooveScribe-Einbindung
- Export von Monatsdaten als CSV

## Voraussetzungen

- Windows
- .NET 10 SDK
- Microsoft Edge WebView2 Runtime
